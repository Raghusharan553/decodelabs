
## task1-TODO.py
### Project Title
**TO-DO List (Console App)**

### Description
A simple command-line to-do list application that lets you add tasks and view them.
- Displays a menu with options to add tasks, view tasks, or exit.
- Stores tasks in memory during the program run.
- Prints tasks with auto-incrementing IDs.

### How to Run
```bash
python task1-TODO.py
```

## task2-Expense_Tracker.py
### Project Title
**Expense Tracker (Console App)**

### Description
Tracks the sum of expenses entered by the user and prints totals.

## task3-pw_generator.py
### Project Title
"RANDOM PASSWORD GENERATOR"

### Description
Generates a strong password and estimates its entropy/security level.
- Creates a password using lowercase, uppercase, digits, and symbols.
- Calculates entropy using `E = L × log2(R)` and labels strength (Weak/Moderate/Strong/Very Strong).
- Validates password length input (100–500).

### How to Run
```bash
python task3-pw_generator.py
```

## task4_Quiz_Game.py
### Project Title
**Quiz Game (Console App)**

### Description
A command-line quiz game that asks 5 multiple-choice questions and calculates your score.
- Displays a question with multiple options.
- Accepts user input for the answer and checks if it matches the correct option.
- Shows the final score out of 5.
- Prompts to play again until you type anything other than `yes`.

### How to Run
```bash
python task4_Quiz_Game.py
```
